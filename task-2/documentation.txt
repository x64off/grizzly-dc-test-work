Папка assets содержит: css и js файлы
../css/popup.css - стиль для cookies popup
../css/toast.css - стиль всплывающего окна с названием страны
../js/task.js - javascript для запроса к php и для cookies popup
../js/toast.js  - скрипт для вызова всплывающего окна с названием страны
Папка templates - html шаблон для главной (единственной) страницы сайта (скрипта)
index.php - точка входа в скрипт и логику определения номера
